# Sambhrama Khushi - Portfolio

A minimalist, parallax-scrolling portfolio website showcasing my experience as a Machine Learning & AI Engineer. Built with a focus on performance, clean aesthetics, and smooth cinematic transitions.

## Features

- **Minimalist & Brutalist Design**: High-contrast aesthetic with ample whitespace and film grain overlay.
- **Cinematic Parallax Scrolling**: Smooth scroll-snap and parallax effects using Framer Motion.
- **Dynamic Header**: Sticky header that morphs into a floating pill design on scroll.
- **Interactive Elements**: Hover reveals for certificates, project cards, and a dynamic skill tag cloud.
- **Dark/Light Mode**: Seamless theme toggling.
- **Responsive**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

- **Framework**: [React](https://reactjs.org/) (Vite)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sambhrama754/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`.

### Building for Production

To create a production-ready build:

```bash
npm run build
```

This will generate a `dist` folder containing the optimized static assets, ready to be deployed to any static hosting service (GitHub Pages, Vercel, Netlify, etc.).
